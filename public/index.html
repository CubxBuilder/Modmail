Anleitung für Moderatoren
1. Ticket-Übersicht Sobald ein User schreibt, erscheint ein neuer Forum-Post.
2. User Infos Innerhalb des Posts erfahren die Moderatoren alle nötigen Infos über den User und dessen geschriebenen Nachrichten.
3. Ticket-Management Den Moderatoren werden drei Buttons bereitgestellt, welche sie für eine bessere Moderation nutzen können.

Claim Ticket: Beanspruche ein Ticket und mache damit dem Team klar, dass du dich darum kümmerst. (Nur für Moderation sichtbar)

User warnen: Sendet eine vorgefertigte Warn Nachricht an den User, die ihn darauf hinweist sich an die Gesprächsregeln zu halten.

Ticket schließen: Schließt den Post und unterbricht den Nachrichtenverlauf (unwiderruflich)
4. NachrichtenNachrichten werden beim Abschicken unwiderruflich an den User weitergeleitet.
